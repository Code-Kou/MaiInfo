# 🎵 MaiInfo
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

MaiInfo is a community-driven leaderboard and player profile platform for the arcade rhythm game **maimai**.

---

## 📖 Description

MaiInfo allows players to:

- View leaderboards
- Search and view player profiles
- Add and manage friends
- Track stats and performance

This project is built as a passion project for the maimai community.

---

## ⚙️ How It Works

MaiInfo uses community-made APIs to retrieve player data.

- Data is fetched from external/community sources
- Users can opt-in to make their profiles visible
- The platform displays player stats in a clean and accessible way

> This project does NOT directly interact with official game servers.

---

## 🔐 Disclaimer

MaiInfo is a fan-made project and is **not affiliated with SEGA**.

- No account credentials are accessed or stored
- No modifications are made to official game data
- All data is either public or user-consented
